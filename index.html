The Prophitt Website

Timeline: May 2026

This website is a multi-page artist portfolio and music archive designed for hip-hop producer and DJ The Prophitt. he site combines a cinematic, minimalist aesthetic with interactive functionality to showcase music releases, live performances, and artist identity.

It prioritizes responsive design, smooth user interaction, modern music-platform-inspired UI/UX, and scalable content organization for future releases and performances. The project is built using HTML, CSS, and JavaScript.

The homepage features a full-screen parallax hero image with fixed typography and smooth fade-in transitions leading into an About Me section and contact/social links. A fixed navigation bar allows seamless movement between pages including Discography and Experience.

SEARCH LOGISTICS:

The Discography page includes searchable and sortable album collections with multiple layout views, detailed album pages with full tracklists and streaming links, and a dynamic song-search system that links individual tracks back to their respective album pages. The Experience page mirrors this functionality with searchable and sortable event listings displaying performance history, venues, dates, and descriptions.

There is also options to organize the albums via album cover with descriptions, or by a grid of album covers.
